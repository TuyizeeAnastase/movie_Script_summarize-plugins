## Project setup
---

### Start Up

- Clone the repository
- Install packages by running `npm install`

### Dotenv setup

- Create `.env` in project root directory
- add openAI Key `OPENAI_API_KEY="KEY_VALUE"`


### Running server

- Run `npm run dev` to run server local
